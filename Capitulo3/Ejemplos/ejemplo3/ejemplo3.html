Java How to Program

De Java How to Program, 9/e:

Caracteríticas:

  • Extensa cobertura de los fundamentos: dos capítulos sobre instrucciones de control.
  • Enfoque en ejemplos del mundo real.
  • Conjunto de ejercicios Hacer la diferencia.
  • Introducción desde los primeros capítulos a las clases, objetos, métodos y cadenas.
  • Manejo de exceociones integrado.
  • Archivos, flujos y serialización de objetos.
  • Seeciones modulares opcionales sobre el lenguaje y características de la biblioteca de la nueva versión Java SE 7.
  • Otros temas: Recursividad, búsqueda, ordenamiento, colecciones genéricas, genéricos, estructuras de datos, applets,
    multimedia multihilos, base de datos,/JDBC™, desarrollo de aplicaciones Web, servicios Web, y un caso de estudio
    opcional de Diseño orientado a objetos con un ATM.
